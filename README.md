## iTerm
Note about Clear Buffer keybind (Cmd-k by default): I've assigned Cmd-k to switching to the split pane above the current one (along with Cmd-h, Cmd-j, Cmd-l for the other directions). Therefore, I had to select a different keybind for clearing the buffer and I chose Cmd-Shift-k, which seems like the closest one.

Note that I could have chosen Ctrl instead of Cmd for pane navigation, but those keybinds are reserved for vim window navigation.
